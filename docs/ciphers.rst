What Encryptions does Ciphey support?
====================================

Basic Encryptions
------------------
* Caesar Cipher
* Vigenere Cipher
* Pig Latin
* Reverse Cipher
* Transposition Cipher

Encodings
----------
* Base16
* Base32
* Base64
* Base85
* Base85 Ascii
* Base62 Bitcoin
* Base62 Ripple
* Base62 (with the char set link shorteners use)
* Ascii
* Hexadecimal
* Binary
* Morse Code

Hashes
-------
* MD5
* SHA-1
* SHA-256
* SHA-384
* SHA-512
